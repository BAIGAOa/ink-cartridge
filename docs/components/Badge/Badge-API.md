# Badge

Small colored label with a background color.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | **required** | Badge text. |
| `color` | `string` | `'cyan'` | Background color (foreground is always black). |

## Best Practice

```tsx
<Badge color="green">NEW</Badge>
```
