# Dev Tool

## Why

Debugging keyboard behavior in a terminal UI is hard — you can't open browser DevTools. The dev tool renders as a modal overlay showing real-time screen path, overlay/modal state, focus targets, and full keyboard binding details.

## API Index

| API | Purpose |
|-----|---------|
| [openDevTool](./openDevTool-API.md) | Open the dev panel |
| [closeDevTool](./closeDevTool-API.md) | Close the dev panel |
