# blots-editor

> A lightweight Markdown editor for the terminal — built on [ink-cartridge](https://github.com/BAIGAOa/ink-cartridge).

## About

blots-editor is both a **real product** and a **real-world testbed** for ink-cartridge. Every screen, keyboard shortcut, and overlay exercises the framework's screen management and keyboard engine in a concrete application — catching edge cases that unit tests alone can't.

Built with [Ink](https://github.com/vadimdemedes/ink) + [React](https://react.dev).

## Install

```bash
npm install -g blots-editor
```

## Usage

```bash
blots-editor              
```
